# Anmeldung

!!! tip "Download der Anleitung"
     [Link zum Download der Anleitung](https://github.com/gz-bad-erzland-p2/Dokumentation/raw/master/docs/assets/Support-Anleitungen/Arbeitsplatzbuchung.pdf)


### 1. Klicken Sie im Menü auf "Anmelden", um zur Anmeldeseite zu gelangen.
![Step 1 screenshot](https://images.tango.us/workflows/77342c59-8ae2-4df8-b994-c182f6606d57/steps/9c4f9167-6440-4662-9c36-2606c4dae204/dc9e5db7-8bc1-43da-a4f8-a355d981b091.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)


### 2. Geben Sie nun die E-Mail-Adresse ein, mit der Sie sich registriert haben. Wenn Sie sich noch nicht registriert haben, führen Sie dies bitte zuerst durch.
![Step 2 screenshot](https://images.tango.us/workflows/77342c59-8ae2-4df8-b994-c182f6606d57/steps/63bfa214-1a77-456f-a14b-c19a54da91f9/46af94c0-2ebc-42bc-9d74-36aaa2f7a87f.png?crop=focalpoint&fit=crop&fp-x=0.4998&fp-y=0.4804&fp-z=1.4966&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)


### 3. Geben Sie nun das Passwort ein, das Sie bei Ihrer Registrierung festgelegt haben.
![Step 3 screenshot](https://images.tango.us/workflows/77342c59-8ae2-4df8-b994-c182f6606d57/steps/1eef979f-fcfe-408f-9c0b-fb43f0babd18/fea215de-9b5d-4719-b69a-32c3640b3837.png?crop=focalpoint&fit=crop&fp-x=0.4998&fp-y=0.5414&fp-z=1.4966&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)


### 4. Bestätigen Sie nun Ihre Eingabe, indem Sie auf "Anmelden" klicken. Anschließend sind Sie in Ihrem Konto angemeldet.
![Step 4 screenshot](https://images.tango.us/workflows/77342c59-8ae2-4df8-b994-c182f6606d57/steps/e1e86c8a-47f9-47fd-9c85-f4c7e533b3c5/4c219da0-5070-42ef-a4b1-6158250c93f8.png?crop=focalpoint&fit=crop&fp-x=0.4998&fp-y=0.6032&fp-z=2.6564&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)
