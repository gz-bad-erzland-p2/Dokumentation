# Registrierung

!!! tip "Download der Anleitung"
        [Link zum Download der Anleitung](../../../../assets/Support-Anleitungen/Registrieren.pdf)

### 1. Öffnen Sie das Menü oben rechts und klicken Sie auf "Registrieren".
![Step 1 screenshot](https://images.tango.us/workflows/c482c676-b72d-43d5-8224-7fef821a72e8/steps/d32db22c-6202-4761-acce-c9fb68f4fcc2/828b0286-a34a-4ea0-b0c9-6bea87bc0dff.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)


### 2. Füllen Sie das Anmeldeformular mit den angeforderten Informationen aus.
![Step 2 screenshot](https://images.tango.us/workflows/c482c676-b72d-43d5-8224-7fef821a72e8/steps/a9287d17-7e54-4133-9324-25e2df3d6c16/526f1687-ce31-4d65-b7c1-e3d0346646f8.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.4680&fp-z=1.0727&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)


### 3. Stimmen Sie den Datenschutzbestimmungen und den Allgemeinen Geschäftsbedingungen zu.
![Step 3 screenshot](https://images.tango.us/workflows/c482c676-b72d-43d5-8224-7fef821a72e8/steps/35348c4e-35a3-4308-8b2e-a835c9220b79/c1e3f702-f511-4b68-808b-26042da038fc.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.5000&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)


### 4. Klicken Sie auf "Registrieren", um Ihre eingegebenen Daten zu überprüfen. Wenn alle Angaben korrekt sind, ist Ihre Registrierung abgeschlossen. Wenn einige Informationen als fehlerhaft gekennzeichnet sind, korrigieren Sie diese und wiederholen Sie den Schritt.
![Step 4 screenshot](https://images.tango.us/workflows/c482c676-b72d-43d5-8224-7fef821a72e8/steps/eed370b8-886b-42a3-978e-19193eeef334/6aa12503-13a8-4cd5-b721-cdbf8ca0a3fc.png?crop=focalpoint&fit=crop&fp-x=0.5000&fp-y=0.7402&fp-z=2.7070&w=1200&blend-align=bottom&blend-mode=normal&blend-x=800)
